Redirects can be imported in bulk using either .xlsx, .xls, .csv, .txt, or .tsv files.

Files with a .txt extension are treated as tab-delimited.

Import files should be in the following format (illustrated as CSV):

        
 OldUrl,NewUrl
 /old-url,/new-url
 /old-url1.aspx,https://example.com
 /old-url/,/

The OldUrl and NewUrl columns must exist, but they need not necessarily be in this order. The OldUrl value must be prefixed with a /. If NewUrl is on the same host, it should also be prefixed with /, otherwise should be prefeixed with the scheme (http:// or https://).
